# Releases

## Versioning

Jarvis Dashboard uses **unified versioning** — one version number covers all platforms since JavaScript changes affect all of them simultaneously.

Version format: `vMAJOR.MINOR.PATCH`

- **MAJOR** — Breaking changes to config format, module contract, or API
- **MINOR** — New features, widgets, or platform support
- **PATCH** — Bug fixes, performance improvements, documentation
