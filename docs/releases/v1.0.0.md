# v1.0.0 — Initial Release

**Date:** March 2025

## Overview

First public release of Jarvis Dashboard — a modular AI command center for monitoring Claude Code sessions across 4 platforms.

## Platforms

- **Obsidian Desktop** — Full dashboard via DataviewJS
- **Obsidian Mobile** — Voice-only interface with companion server
- **macOS (Tauri 2.0)** — Native desktop app with Rust backend
- **iOS/iPadOS (SwiftUI)** — Native mobile app with WKWebView bridge

## Features

### Core
- Cross-platform shared JavaScript codebase (no build step)
- Platform adapter abstraction (Obsidian, Tauri, WKWebView)
- Config-driven layout system with 4-file cascade merging
- Module loading via `new Function("ctx", code)` pattern
- Pause/resume system (visibility API)
- MutationObserver-based cleanup

### Widgets (13)
- **Header** — Title, live clock, status indicator
- **Voice Command** — Full voice/text interaction with Claude Code
- **Live Sessions** — Real-time Claude session monitoring
- **Agent Cards** — AI agent status visualization
- **Focus Timer** — Pomodoro timer with vault logging
- **Quick Capture** — Note capture with voice dictation
- **Activity Analytics** — 30-day usage heatmaps and charts
- **System Diagnostics** — Aggregate usage statistics
- **Communication Link** — Terminal/editor launcher
- **Quick Launch** — Configurable app/URL bookmarks
- **Mission Control** — Dashboard navigation
- **Recent Activity** — Recently modified vault files
- **Footer** — Dashboard footer

### Voice System
- Three TTS engines: Piper (neural), macOS Say, browser speechSynthesis
- Speech-to-text via whisper-cpp
- Multi-language support with auto-detection
- Sentence-boundary streaming for low-latency TTS
- Markdown stripping for natural speech
- Mute toggle with persistent state

### Interactive Mode
- Permission cards for tool approval (Bash, Write, Edit)
- Auto-approve list for safe tools (Read, Glob, Grep)
- Question cards for Claude inquiries
- Configurable timeouts
- Voice response to questions

### Session Management
- Multi-session with tab bar
- Drag & drop tab reordering
- Session persistence to `~/.claude/jarvis-sessions.json`
- Max 10 sessions, auto-prune oldest idle
- Per-session terminal output

### Companion Server
- Dual WebSocket: WSS :7777 (TLS) + WS :7778 (local)
- TLS with self-signed CA certificate chain
- Token authentication (timing-safe comparison)
- Rate limiting, connection limits, idle timeout
- Claude CLI process management
- Audio transcription pipeline (ffmpeg → whisper-cli)
- TTS synthesis and streaming (Piper + Say engines)
- LaunchAgent auto-start support
- Tailscale VPN support

### macOS App (Tauri)
- 17 Rust commands for file I/O, process, vault, OS operations
- Native menu bar
- Built-in audio transcription
- Vault folder picker on first launch

### iOS App
- SwiftUI with WKWebView bridge
- AVAudioEngine recording
- iOS Keychain credential storage
- Settings screen for server configuration
- Certificate trust handling

### Analytics
- 30-day statistics with caching
- Per-day activity heatmap
- Model usage breakdown (Opus/Sonnet/Haiku)
- Peak hours analysis
- Cost estimation with configurable pricing

### Performance
- GPU compositing for animations
- Background thread processing
- Centralized pause system
- Configurable polling intervals
- Deferred widget rendering (`contentVisibility: auto`)
- Stats caching

## Configuration

- 4-file cascade: config.example.json → config.json → config.local.json → platform overrides
- 15 theme colors
- Configurable layout array
- JARVIS personality system with placeholders
- Agent registry via YAML frontmatter
- Per-language TTS model and parameter overrides
